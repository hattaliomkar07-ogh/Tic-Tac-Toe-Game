# Tic Tac Toe Game

## Project Overview

This project is a simple console-based Tic Tac Toe game developed using C++. The game demonstrates the use of arrays, loops, conditional statements, and basic game logic.

## Aim

To develop a Tic Tac Toe game in C++ that allows two players to play interactively through the console.

## Features

- Dynamic game board display
- Two-player gameplay
- Win detection
- Draw detection
- Replay option
- Uses arrays and loops
- Console-based interface

## Technologies Used

- C++
- Arrays
- Loops
- Conditional Statements

## How to Run

1. Open the project in any C++ IDE.
2. Compile the source file.
3. Run the program.
4. Follow the on-screen instructions to play.

## Expected Outcome

The game allows two players to play Tic Tac Toe while demonstrating fundamental programming concepts such as arrays, loops, and conditional logic.
